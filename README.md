# 页面设计框架

这个仓库包含了三个页面设计框架的HTML文件：

- 店铺页装修框架.html
- 活动页装修框架.html
- 详情页设计框架.html

## 访问链接

启用 GitHub Pages 后，可以通过以下三个独立链接访问：

1. **店铺页**：`https://[你的用户名].github.io/[仓库名]/店铺页装修框架.html`
2. **活动页**：`https://[你的用户名].github.io/[仓库名]/活动页装修框架.html`
3. **详情页**：`https://[你的用户名].github.io/[仓库名]/详情页设计框架.html`

详细的上传和配置步骤请查看 [上传指南.md](./上传指南.md)
