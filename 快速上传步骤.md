# 快速上传步骤 - JDframe 仓库

## 步骤 1：在 GitHub 上创建仓库

1. 访问 https://github.com/new
2. Repository name: **JDframe**
3. Description: 页面设计框架
4. 选择 **Public**（公开）
5. **不要**勾选任何初始化选项（README、.gitignore、license）
6. 点击 **Create repository**

## 步骤 2：执行推送（我会帮你完成）

创建好仓库后，告诉我，我会立即帮你推送代码！

或者你可以直接运行：
```bash
./推送命令.sh
```

## 步骤 3：启用 GitHub Pages

1. 访问 https://github.com/Snow968896/JDframe/settings/pages
2. Source 选择：**main** 分支，**/ (root)** 文件夹
3. 点击 **Save**
