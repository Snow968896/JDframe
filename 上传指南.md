# GitHub 上传和发布指南

## 步骤 1：在 GitHub 上创建新仓库

1. 登录 GitHub (https://github.com)
2. 点击右上角的 "+" 号，选择 "New repository"
3. 填写仓库信息：
   - Repository name: 例如 `page-design-frameworks`（或你喜欢的名字）
   - Description: 页面设计框架
   - 选择 **Public**（公开仓库才能使用免费的 GitHub Pages）
   - **不要**勾选 "Initialize this repository with a README"
4. 点击 "Create repository"

## 步骤 2：推送代码到 GitHub

在终端中执行以下命令（将 `YOUR_USERNAME` 和 `YOUR_REPO_NAME` 替换为你的实际信息）：

```bash
cd "/Users/snowsun/Desktop/25十二月/设计/框架"

# 添加远程仓库（替换下面的URL为你的仓库地址）
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# 推送代码
git branch -M main
git push -u origin main
```

或者如果你使用 SSH：

```bash
git remote add origin git@github.com:YOUR_USERNAME/YOUR_REPO_NAME.git
git branch -M main
git push -u origin main
```

## 步骤 3：启用 GitHub Pages

1. 在 GitHub 仓库页面，点击 **Settings**（设置）
2. 在左侧菜单中找到 **Pages**（页面）
3. 在 "Source" 部分：
   - 选择分支：**main**
   - 选择文件夹：**/ (root)**
4. 点击 **Save**（保存）

## 步骤 4：访问你的网页

等待几分钟后（通常 1-2 分钟），你的三个网页就可以通过以下**三个独立链接**访问：

### 三个独立的访问链接：

1. **店铺页链接**：
   ```
   https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/店铺页装修框架.html
   ```

2. **活动页链接**：
   ```
   https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/活动页装修框架.html
   ```

3. **详情页链接**：
   ```
   https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/详情页设计框架.html
   ```

**示例**（假设用户名为 `snowsun`，仓库名为 `page-design-frameworks`）：
- 店铺页：`https://snowsun.github.io/page-design-frameworks/店铺页装修框架.html`
- 活动页：`https://snowsun.github.io/page-design-frameworks/活动页装修框架.html`
- 详情页：`https://snowsun.github.io/page-design-frameworks/详情页设计框架.html`

## 注意事项

- 如果文件名包含中文，URL 中会显示为编码后的形式，这是正常的
- GitHub Pages 更新可能需要几分钟时间
- 确保仓库是 **Public**（公开）的，否则 GitHub Pages 无法访问
